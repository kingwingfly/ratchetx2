E2EE chat TUI client.

```sh
An E2EE chat client.

Usage: e2ee_chat_client [OPTIONS] <SERVER_ADDR>

Arguments:
  <SERVER_ADDR>  The E2EE chat server address

Options:
  -c, --ca <CA>  The path to self-sign CA pem (only for https and test)
  -h, --help     Print help
  -V, --version  Print version
```
